¿Por qué decidimos hacer una página web con chatbot como proyecto?

Hicimos una página web, ya que algunos de nuestro equipo, sabían manejar un poco HTML y CSS. Y nos pareció interesante e innovador.

Además de esto, también nos pareció interesante hacer el chatbot porque el año pasado no vimos ningún otro proyecto en la feria de ciencias que presentara un chatbot.

Lenguajes usados y programas

    Apps

  • XAMPP: Es una herramienta de desarrollo que te permite probar tu desarrollo web basado en PHP en tu propio ordenador sin necesidad de tener acceso a internet

  • Visual Studio code: es un entorno de desarrollo integrado (IDE) que proporciona herramientas y características para escribir y depurar código

    • Lenguajes

  • HTML: lenguaje de marcación que sirve para definir el contenido de las páginas web(estructura de la página).

  • CSS: Lenguaje de diseño gráfico para definir y crear la presentación de un documento estructurado escrito en un lenguaje de marcado
  • JS:lenguaje de programación interpretado, dialecto del estándar ECMAScript. Se define como orientado a objetos, basado en prototipos, imperativo, débilmente tipado y dinámico.

¿Cómo creamos los chatbots?

Los chatbots los creamos usando botsonic que está basado en la AI sonic, Creamos cinco bots diferentes con cinco emails diferentes, uno para cada sistema, los personalizamos y los insertamos a la web con un script de JS que nos proporcionaba el chatbot.
