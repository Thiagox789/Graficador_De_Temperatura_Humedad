    Sistema Endocrino

    El sistema endocrino es un complejo sistema del cuerpo humano compuesto por glándulas endocrinas que producen y liberan hormonas en la corriente sanguínea. Estas hormonas actúan como mensajeros químicos, regulando diversas funciones fisiológicas y manteniendo el equilibrio interno del organismo.

    Sistema Digestivo

    El sistema digestivo es un conjunto de órganos encargados de procesar los alimentos para obtener nutrientes esenciales y eliminar los residuos.

    Leer más

    03

    Sistema Circulatorio

    El sistema circulatorio es una red de vasos sanguíneos, como arterias y venas, que transporta la sangre por todo el cuerpo. La sangre lleva oxígeno y nutrientes a las células, y recoge desechos para ser eliminados.

    Leer más

    04

    Sistema Respiratorio

    El sistema respiratorio es responsable de llevar oxígeno a las células del cuerpo y eliminar dióxido de carbono. Incluye órganos como la nariz, la tráquea y los pulmones.

    Leer más

    05

    Sistema Nervioso

    El sistema nervioso coordina y controla las actividades del cuerpo. Incluye el cerebro, la médula espinal y los nervios.
